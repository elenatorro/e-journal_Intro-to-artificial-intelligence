# Search Tree

```
          ____[A]____
         /     |     \
      [ ]     [ ]     [ ]
     /   \   /   \   /   \
  [ ]   [B] [ ] [ ] [ ]  [ ]
```

How many nodes will be expanded from A to B?

Left-To-Right
* Breadth First => 6

```
          ____[1]____
         /     |     \
      [2]     [3]     [4]
     /   \   /   \   /   \
  [5]   [6] [ ] [ ] [ ]  [ ]
```

* Depth First => 4

```
          ____[1]____
         /     |     \
      [2]     [ ]     [ ]
     /   \   /   \   /   \
  [3]   [4] [ ] [ ] [ ]  [ ]
```
